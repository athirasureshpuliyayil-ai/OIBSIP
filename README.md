# OIBSIP Projects

This repository contains three Python projects:

## 1. Password Generator
A Python program to generate secure passwords.

 Advanced Random Password Generator

 Features:
- GUI interface with Tkinter
- Generate customizable passwords (uppercase, lowercase, digits, symbols)
- Copy password to clipboard
- Check password strength
- Save passwords securely in JSON file

 Installation:
1. Install required packages:
   pip install -r requirements.txt

2. Run the app:
   python main.py

 How to Use:
1. Set password length and select options (uppercase, lowercase, digits, symbols)
2. Click "Generate Password"
3. Copy to clipboard or save password to file
4. Check password strength visually


## 2. Weather App
A Python program to fetch weather data from an API.
# Weather App (No API Key Required)

This project uses the Open-Meteo API which is 100% free and does NOT require any API key.

 Features:
- GUI built using Tkinter
- Instantly fetch weather for any city
- Uses Open-Meteo free weather API
- Shows temperature, wind speed, and condition
- No signup, no billing, no API key

 Install Required Packages:
pip install -r requirements.txt

 Run the App:
python main.py

 How It Works:
1. Enter a city name (e.g., Delhi, Paris, Tokyo)
2. Click "Get Weather"
3. See weather results instantly


## 3. BMI Calculator
A Python program to calculate Body Mass Index (BMI).

 Advanced BMI Calculator

 Features:
- GUI interface using Tkinter
- Input validation (height & weight must be positive)
- Save BMI history in CSV
- Visualize BMI history with matplotlib graphs
- Personalized health tips based on BMI
- Multi-user support

 Installation:
1. Install required packages:
   pip install -r requirements.txt

2. Run the app:
   python main.py

 How to Use:
1. Enter your name, height (cm), and weight (kg)
2. Click "Calculate BMI" to see BMI, category, and health tips
3. Click "Show BMI History" to see a bar chart of all users' BMI

Run the `.py` files using Python 3:

```bash
python main.py
